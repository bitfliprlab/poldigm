# 📄 11_결과지_텍스트_맵핑_문서

## 1. 문서 개요

본 문서는 폴다임(Poldigm) 테스트 완료 후, 백엔드에서 산출된 최종 결과 코드(예: `CTMO-S`)를 프론트엔드 결과 페이지(UI)에 출력하기 위해 필요한 모든 텍스트 및 속성 데이터를 정의한 맵핑 테이블입니다. 프론트엔드 개발자는 본 문서의 JSON/DB 구조를 참고하여 결과 렌더링 로직을 구현합니다.

## 2. 결과지 화면 구성 요소 (Data Schema)

결과 페이지는 도출된 **16가지 알파벳 조합**과 **S/M 태그**에 따라 동적으로 조립됩니다.

- **`type_code`**: 16가지 기본 성향 코드 (예: CTMO, CPEL 등)
    
- **`title`**: SNS 바이럴을 유도하는 직관적이고 감성적인 페르소나 타이틀
    
- **`subtitle`**: 해당 성향을 요약하는 한 줄 카피
    
- **`character_img`**: 맵핑될 캐릭터 일러스트 에셋명 (디자인팀 전달 예정)
    
- **`desc_S` (강경파 텍스트)**: 태그가 `-S`일 때 출력될 확고한 이념형 상세 설명
    
- **`desc_M` (온건파 텍스트)**: 태그가 `-M`일 때 출력될 실용주의/타협형 상세 설명
    
- **`chemistry_best`**: 환상의 파트너 (가장 잘 맞는 유형)
    
- **`chemistry_worst`**: 파국의 앙숙 (가장 안 맞는 유형)
    

## 3. 핵심 유형별 텍스트 맵핑 테이블 (일부 발췌)

_※ 16개 전체 유형 중 가장 대비되는 대표 6개 유형의 맵핑 데이터입니다. 전체 데이터는 개발용 JSON 파일로 별도 제공됩니다._

### 1) CTMO (공동체·전통·능력·질서)

- **타이틀 (`title`)**: 냉철한 시스템 설계자
    
- **한 줄 카피 (`subtitle`)**: "무너진 규칙을 세우고, 검증된 실력자에게 보상하라."
    
- **강경파 설명 (`desc_S`)**: 당신은 사회가 올바르게 굴러가기 위해 강력한 통제와 엄격한 규칙이 필수라고 믿는 철저한 시스템 설계자입니다. 검증되지 않은 급진적 변화를 경계하며, 압도적인 능력을 증명한 자만이 권력과 보상을 독점하는 것이 가장 완벽한 공정이라 생각합니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 안정적인 사회 시스템을 중시하는 합리적 보수주의자입니다. 전통과 질서를 존중하지만, 시대의 흐름에 따라 낡은 규칙은 서서히 고쳐나갈 줄 아는 유연함을 갖췄습니다.
    
- **궁합 매칭**:
    
    - **Best**: ITML (고독한 자유시장 개척자)
        
    - **Worst**: CPEL (광장의 따뜻한 연대자)
        

### 2) CPEL (공동체·진보·평등·자유)

- **타이틀 (`title`)**: 광장의 따뜻한 연대자
    
- **한 줄 카피 (`subtitle`)**: "아무도 소외되지 않는, 자유롭고 평등한 세상을 향해."
    
- **강경파 설명 (`desc_S`)**: 당신은 기득권의 견고한 벽을 부수고, 모두가 동등한 출발선에 설 수 있도록 사회를 전면 개조해야 한다고 믿는 열정적 혁명가입니다. 누군가의 희생을 밟고 올라서는 엘리트주의를 혐오하며, 연대와 인권을 최우선 가치로 삼습니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 사회적 약자를 배려하고 평등을 지향하는 따뜻한 마음을 가졌습니다. 급진적인 개혁보다는 현실 가능한 복지 정책과 제도적 개선을 통해 서서히 더 나은 세상을 만들고자 하는 실용적 진보주의자입니다.
    
- **궁합 매칭**:
    
    - **Best**: IPEL (자유로운 보헤미안 혁명가)
        
    - **Worst**: CTMO (냉철한 시스템 설계자)
        

### 3) ITML (개인·전통·능력·자유)

- **타이틀 (`title`)**: 고독한 자유시장 개척자
    
- **한 줄 카피 (`subtitle`)**: "내 살길은 내가 찾는다. 국가의 간섭은 사양합니다."
    
- **강경파 설명 (`desc_S`)**: 당신은 개인의 능력과 자유를 억압하는 국가의 모든 규제와 세금을 죄악으로 여기는 확고한 자유방임주의자입니다. 세상은 적자생존이며, 철저한 경쟁에서 승리한 자가 모든 것을 얻는 것이 자연의 섭리라고 믿습니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 시장의 자유와 개인의 성취를 존중하는 합리적 개인주의자입니다. 국가의 개입을 최소화하길 원하지만, 범죄 예방이나 최소한의 국방 등 필수적인 공권력의 필요성은 인정하는 현실주의자입니다.
    
- **궁합 매칭**:
    
    - **Best**: CTMO (냉철한 시스템 설계자)
        
    - **Worst**: CPEL (광장의 따뜻한 연대자)
        

### 4) IPEL (개인·진보·평등·자유)

- **타이틀 (`title`)**: 자유로운 보헤미안 혁명가
    
- **한 줄 카피 (`subtitle`)**: "낡은 족쇄를 끊어라. 내 인생은 나의 것!"
    
- **강경파 설명 (`desc_S`)**: 당신은 꼰대스러운 전통과 집단주의를 혐오하는 아나키스트에 가깝습니다. 개인의 표현의 자유가 그 어떤 공익보다 중요하며, 사회적 불평등 역시 국가의 통제가 아닌 개인들의 자발적이고 자유로운 연대를 통해 해결할 수 있다고 믿습니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 얽매이는 것을 싫어하고 다양성을 존중하는 트렌디한 개인주의자입니다. 낡은 관습을 타파하는 것을 좋아하지만, 사회적 안정망을 위해 어느 정도의 공동체적 합의와 세금 납부는 수용할 줄 아는 융통성을 지녔습니다.
    
- **궁합 매칭**:
    
    - **Best**: CPEL (광장의 따뜻한 연대자)
        
    - **Worst**: CTMO (냉철한 시스템 설계자)
        

### 5) IPMO (개인·진보·능력·질서)

- **타이틀 (`title`)**: 실리콘밸리의 혁신 몽상가
    
- **한 줄 카피 (`subtitle`)**: "기술과 데이터로 세상을 해킹하고 효율을 극대화하라."
    
- **강경파 설명 (`desc_S`)**: 당신은 오직 논리와 효율, 압도적 기술력만이 세상을 구원한다고 믿는 테크노크라트(기술관료제)입니다. 감성적인 평등주의나 낡은 전통을 경멸하며, 초격차 능력을 가진 소수의 천재들이 강력한 시스템을 구축해 우매한 대중을 이끌어야 한다고 생각합니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 혁신과 능력주의를 사랑하는 스마트한 현대인입니다. 데이터 기반의 합리적인 의사결정을 중시하며, 비효율적인 옛날 방식은 과감히 버리되 극단적인 성과주의로 인한 피로감은 경계하는 균형 감각을 가졌습니다.
    
- **궁합 매칭**:
    
    - **Best**: CTMO (냉철한 시스템 설계자)
        
    - **Worst**: CTEL (울타리 속의 다정한 수호자)
        

### 6) CTEL (공동체·전통·평등·자유)

- **타이틀 (`title`)**: 울타리 속의 다정한 수호자
    
- **한 줄 카피 (`subtitle`)**: "전통의 가치를 지키며, 다 함께 나누는 따뜻한 마을."
    
- **강경파 설명 (`desc_S`)**: 당신은 종교나 향토적 가치 같은 전통적 공동체를 깊이 사랑하며, 그 안에서 구성원들이 소외됨 없이 평등하게 자원을 나누어야 한다고 믿는 공동체주의자입니다. 다만 외부의 급진적 변화나 검증되지 않은 새로운 사상에 대해서는 강한 배타성을 보일 수 있습니다.
    
- **온건파 설명 (`desc_M`)**: 당신은 이웃 간의 정과 전통적인 예의를 중시하는 평화주의자입니다. 치열한 무한 경쟁보다는 다 같이 적당히 나누어 먹는 삶을 선호하며, 갈등을 싫어해 중재자 역할을 자처하는 포용력을 지녔습니다.
    
- **궁합 매칭**:
    
    - **Best**: CPEL (광장의 따뜻한 연대자)
        
    - **Worst**: IPMO (실리콘밸리의 혁신 몽상가)
